# Session Review: GTM and privacy v1

Date: 2026-05-14 04:35
Scope: `PUBLIC_GTM_ID` を使う Google Tag Manager 条件付き埋め込みと privacy page 更新
Summary: `BaseLayout` に production + `PUBLIC_GTM_ID` 条件で GTM standard snippet を追加し、`/privacy` に basic analytics 利用の説明を追記した。env なし build では GTM が出ないこと、test ID 付き build では `head` script と `body` 直後 `noscript` が出ることを確認した。

## 1. セッションの目的

- Cloudflare Pages に設定済みの `PUBLIC_GTM_ID` を使って GTM を site に埋め込む
- 公式 install 手順に沿って `head` / `body` へ正しい位置で出力する
- `/privacy` に analytics 利用の最小説明を追加する

## 2. 実際にやったこと

- `site/src/layouts/BaseLayout.astro` に GTM の条件付き出力を追加した
  - 条件:
    - `import.meta.env.PROD`
    - `PUBLIC_GTM_ID` が存在
- `script` は `<head>` 内の上部に追加した
- `noscript` iframe は opening `<body>` 直後に追加した
- `site/src/pages/privacy.astro` を更新し、GTM / GA4 の basic analytics 利用と非対応範囲を追記した
- build を 2 パターンで確認した
  - env なし
  - `PUBLIC_GTM_ID=GTM-TEST123`

## 3. 変更したもの

### Files

- `site/src/layouts/BaseLayout.astro`
- `site/src/pages/privacy.astro`
- `docs/session-reviews/2026-05-14-0435-gtm-privacy-v1.md`

### Commands

- `cd site && npm run build`
  - 目的: env なし build で GTM が出ないことを確認
  - 結果: 成功
- `rg -n "googletagmanager|GTM-" site/dist/index.html site/dist/privacy/index.html`
  - 目的: env なし build の確認
  - 結果: match なし
- `cd site && PUBLIC_GTM_ID=GTM-TEST123 npm run build`
  - 目的: test ID 付き build の確認
  - 結果: 成功
- `rg -n "googletagmanager|GTM-TEST123|ns.html\\?id=GTM-TEST123|gtm.js\\?id=GTM-TEST123" site/dist/index.html site/dist/privacy/index.html`
  - 目的: GTM snippet / noscript iframe の確認
  - 結果: 期待どおり出力

## 4. 確認結果

- `PUBLIC_GTM_ID` 未設定時は GTM snippet は出力されない
- `PUBLIC_GTM_ID` 設定時は GTM standard snippet が出力される
- `script` は `<head>` の上部に出力される
- `noscript` iframe は `<body>` 直後に出力される
- `/privacy` の説明は、既存の静かな文体を大きく崩さずに更新できた
